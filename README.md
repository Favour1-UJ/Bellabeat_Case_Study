# Bellabeat_Case_Study
A data analysis project exploring smart device trends to inform marketing strategy for Bellabeat.
# Executive Summary
The Peak: Users are most active on Tuesdays (**9,492 steps avg**).
The Slump: Activity drops by **14%** on Wednesdays ("The Wednesday Wall")
The Gap: Users spend 11+ hours sedentary, even on active days, and some struggle with sleep latency (time spent awake in bed).
##The Solution: Focus on automated tracking (manual logging is 90% null) and mid-week recovery features.

# The Process
Data Source: Fitbit Fitness Tracker Data (33 Users).
Tools: Power Query (Excel) for ETL & Cleaning.
Integrity Check: Distance columns verified within 0.02 margin of error.
Cleaning: Removed 0-step days; standardized IDs and Dates for merging

# Strategic Recommendation
Trigger "Mid-week Motivation" app alerts
Introduce Sleep Readiness prompts
